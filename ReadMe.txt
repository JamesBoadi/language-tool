To run, unzip and extract to any directory and click on src, 
then index to run (Works best on chrome)

To run offline (if you don't have internet connection) open settings.js with an editor

1. Change the protocol to http

2. Change the endpoint to localhost:8081

Next, open command line and cd to the folder where lantool was extracted

Follow these instructions:

https://dev.languagetool.org/http-server

Have fun.
